System Maintenance: Scheduled Downtime for Version 2.4 Update

calendar_today October 24, 2023
schedule 4 min read

We are pleased to announce the upcoming release of SD Automation Version 2.4. To ensure a smooth transition and implement critical infrastructure improvements, we have scheduled a brief period of system maintenance.

What to Expect

During this window, the main dashboard and API services will be temporarily unavailable. Our team will be working to deploy several highly requested features and performance patches designed to optimize your workflow.

Maintenance Window

  • Start: Saturday, Oct 28, 02:00 AM UTC
  • End: Saturday, Oct 28, 04:00 AM UTC
  • Estimated Duration: 2 Hours

Key Improvements in v2.4

  • check_circle Enhanced Data Visualization: New real-time charting engine for the Reports module providing 2x faster load times.
  • check_circle Advanced Filtering: Added multi-select capability for announcement tags and dashboard widgets.
  • check_circle Security Patch: Implementation of OAuth2.1 standards for improved API authentication security.
  • check_circle Mobile Responsiveness: Improved layout stability for tablet and mobile browser views.

Preview: The new enhanced analytics engine in Version 2.4

Next Steps

No action is required from your side. Your data will remain safe and secure during the maintenance window. We recommend completing any critical tasks at least 30 minutes before the scheduled downtime.

If you have any questions or concerns regarding this update, please reach out to our support team at support@sd-automation.io.
